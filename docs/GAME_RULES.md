# Quiz Territory Game Rules

## Primary Use Case
- A private party quiz game.
- 1 main TV/projector display, 1 main host, up to 1 assistant host.
- 4 to 49 players joining via mobile phones.

## Territory and Adjacency
- Each player starts with one board cell.
- A player may challenge an opponent whose territory is orthogonally adjacent (above, below, left, right).
- Diagonal adjacency is optional but disabled by default.

## Duel Mechanics
- Both players compete in the challenged player's defensive category.
- Players answer verbally; the host manually evaluates (Correct, Wrong, Pass, Skip).
- Only one timer runs at a time.
- If a timer reaches zero, that player is eliminated, and the winner takes their territory and defensive category.
- The winner can continue attacking or return to the random selection pool.

## Game Completion
- The game ends when one player owns the entire board.
- The winner and a top 3 podium are displayed.
